Purple Love Charity Initiative

Role: Frontend Developer | Tech: Payment Logic, DOM Manipulation, n8n
View Live Project ↗

The Challenge

The Purple Love Charity Initiative needed a donation portal that removed friction for potential donors. The main challenge was handling multiple currencies (Naira, Dollars, Pounds) and ensuring that users could easily copy banking details without manual errors.

The Solution

I engineered a "Direct-Action" donation page with a focus on usability:

  • Multi-Currency Support: Designed a layout that clearly separates funding options for local and international donors.
  • Click-to-Copy Logic: Implemented JavaScript functionality allowing users to copy bank account numbers simply by clicking them, reducing user frustration.
  • Backend Automation: Replaced traditional form handling with an n8n automated workflow to instantly notify the charity administrators of new messages or pledges.
  • Responsive Design: Ensured the donation flow was seamless on mobile devices, where 80% of traffic originated.

The Outcome

The updated interface simplified the donation process, resulting in a cleaner user journey and fewer abandoned attempts during the payment phase.
